# Phase 4.5 Kaggle Scaffold

This directory holds the Kaggle-side smoke runner and notebook used for model-heavy
Phase 4.5 validation.

## Contents

- `phase45_kaggle_runner.py` for the shared smoke-run logic.
- `phase45_kaggle_runner.ipynb` as the notebook entry point.
- `kaggle_runtime_setup.md` for the runtime contract.
- `requirements.lock.txt` for any future pinned dependencies.

## Rules

- Keep the notebook aligned with the Python runner.
- Do not add manual notebook-only logic.
- Do not use this directory for official Phase 5 trials.
